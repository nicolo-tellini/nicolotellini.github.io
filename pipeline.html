Data Frame

When you subset a data frame, it does not necessarily return a data frame. For example, if you subset two columns, you get a data frame, but when you try to subset one column, you get a vector:

mtcars[1:5, "mpg"]
## [1] 21.0 21.0 22.8 21.4 18.7

To make sure that we always get a data frame, we have to use the argument drop = FALSE. Now we use the chunk option class.source = "bg-success".

mtcars[1:5, "mpg", drop = FALSE]
##                    mpg
## Mazda RX4         21.0
## Mazda RX4 Wag     21.0
## Datsun 710        22.8
## Hornet 4 Drive    21.4
## Hornet Sportabout 18.7

The data frame is the object that represents a table, where each column can be seen as a vector.

The function data.frame() allows us to organize vectors composing a table.

The length of each column must be the same, like with the matrices, but different columns can store different data type.

# character vector
patients_names = c("Tom","Sarah","Felipe") 

# numeric vector
age = c(18,25,32) 

# factor
test_results = factor(c("+","-","-"),levels = c("+","-")) 

p = data.table::data.table(patients_names,age,test_results)

Every time we approach a data frame, we need to keep in mind its structure and composition.

In some (not so rare) cases we upload datasets from third parties and so it is a good practice become familiar with the data frame we are going to analyse.

Some useful functions are: dim(), str() and summary().

my_data_frame = data.frame(patients_names,age,test_results)

my_data_frame patients_names age test_results 1 Tom 18 + 2 Sarah 25 - 3 Felipe 32 -

dim(my_data_frame) # returns number of rows and number of columns [1] 3 3

str(my_data_frame) ‘data.frame’: 3obs. of 3 variables $ patients_names: Factor w/ 3 levels “Felipe”,“Sarah”,..: 3 2 1 $ age : num 18 25 32 $ test_results : Factor w/ 2 levels “+”,“-”: 1 2 2

summary(my_data_frame) patients_names age test_results Felipe:1 Min. :18.0 +:1
Sarah :1 1st Qu.:21.5 -:2
Tom :1 Median :25.0
Mean :25.0
3rd Qu.:28.5
Max. :32.0
str() returns the structure of an object. It tells us that our object is a data frame with 3 observations (rows) and 3 variables (columns). The first and third columns are factors while the second is a numeric vector. By default data.frame() converts character vectors in factors (note: at the beginning we defined patients_names as a vector not as a factor); to change this we need to specify the option stringsAsFactors = F, inside the function data.frame().

The first column is a factor with 3 levels: “Felipe”, “Sarah” and “Tom” (remember: the order of the levels follows the alphabetical order unless we specify it). The following 3 2 and 1 say us that “Felipe” (the first level) is in the third position of the factor, “Sarah” in the second position and “Tom” in the first one.

The second column is a vector with the ages : 18, 25 and 32.

The third column is a factor with 2 levels: “+”, and “-” ( in this order because we specified it at the beginning with the option levels). The following 1, 2 and 2 say us that “+” is in the first position while “-” is in the second and in the third.

Finally, summary() returns a summary of the content per each column.

The first and the third columns contain characters, so we get the number of occurrences of each element inside them.

The second column contains numeric data so we get a statistical summary of its content: min and max values, median and mean, 1° and 3° quartile.
